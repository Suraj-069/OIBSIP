# 🌡️ Task 3 - Temperature Converter

## Project Overview
A responsive Temperature Converter web application developed as part of the Oasis Infobyte Web Development & Designing Internship. The application converts temperature values between Celsius, Fahrenheit, and Kelvin with input validation and a user-friendly interface.

## Features
- Celsius, Fahrenheit & Kelvin Conversion
- Input Validation
- Instant Results
- Responsive Design
- Formula Reference
- Reset Functionality

## Technologies Used
- HTML5
- CSS3
- JavaScript
